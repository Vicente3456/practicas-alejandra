Conceptos de Ciberseguridad

Concepto Definición
Ciberseguridad Conjunto de medidas, tecnologías y procesos diseñados para proteger sistemas, redes, programas y datos de ataques, daños o accesos no autorizados.
Malware Software malicioso que incluye virus, gusanos, troyanos, ransomware y spyware, creado con el propósito de dañar o robar información.
Phishing Práctica de engañar a las personas para que compartan información confidencial, generalmente mediante correos electrónicos o páginas web falsas.
Firewall Barrera de seguridad que regula el tráfico de red entre redes de confianza y redes externas, bloqueando accesos no autorizados.
Antivirus Programa que detecta, bloquea y elimina software malicioso antes de que pueda dañar el sistema.
Autenticación Multifactor (MFA) Método de seguridad que requiere más de una forma de verificación (como contraseña y código enviado al celular) para acceder a un sistema.
Ingeniería Social Técnicas psicológicas utilizadas por los atacantes para manipular a los usuarios y que revelen información sensible o realicen acciones riesgosas.
Backup Copia de seguridad de los datos que permite recuperarlos en caso de pérdida, fallo del sistema o ataque cibernético.
Criptografía Conjunto de técnicas para proteger la información a través de algoritmos de cifrado, asegurando confidencialidad e integridad.
Vulnerabilidad Debilidad en un sistema o aplicación que puede ser explotada por un atacante para comprometer la seguridad.
